# PKU 3D Navigation System — 北京大学校园3D智能导航系统

本项目实现了一个基于真实校园地理数据的 2D/3D 一体化校园导航系统。用户可以在 2D 地图上选择目的地、管理路线，并一键切换到 3D 场景进行沉浸式导航。系统基于真实校园道路网络，支持多目的地最优路径规划、沿途景点推荐以及途经顺序拖拽调整。

---

## 功能概览

### 1. 2D 地图交互界面
- 建筑点选与列表管理：地图上每个建筑以彩色圆点标识，不同颜色区分未选择、已选、推荐、高亮四种状态。
- 建筑搜索与快速添加：支持输入建筑名称进行检索，输入时自动补全，搜索结果自动居中高亮。
- 偏好驱动的建筑推荐：首次进入时弹出偏好选择对话框，根据用户选择的标签优先级在地图上推荐最多 15 个建筑。
- 沿途景点推荐：在已有路径上搜索偏好标签的兴趣点，按偏离时间排序，支持一键添加并重新规划。
- 路径规划与 3D 跳转：计算最优访问顺序并显示总路程/总时间，路线同时在 2D 地图绘制并传递至 3D 场景。
- 缩略图导航：左上角缩略图显示地图全貌和当前可视区域，点击可快速切换视口。

### 2. 3D 交互界面
- 沉浸式漫游：基于 OpenGL 构建的三维校园场景，支持鼠标拖拽旋转、右键平移、滚轮缩放。
- 真实道路网格与建筑轮廓：道路以实体色带呈现，建筑根据 OSM 轮廓拉伸建模，无轮廓数据时自动回退。
- 导航路径高亮显示：沿真实道路网络的路径以红色线条渲染，建筑顶部显示名称标签。
- 顺序调整面板：覆盖在 3D 场景上的拖拽面板，支持自由调整途经点顺序，调整后双视图同步更新。

### 3. 路径规划引擎
- 基于 Dijkstra 算法的最短路径计算，支持在道路网络上寻找真实路径。
- 多目的地最优顺序求解：使用状态压缩动态规划（TSP）计算访问所有目标点的最优顺序。
- 沿途偏离推荐：按标签筛选 POI，计算插入当前路径的最佳位置，按偏离时间排序。
- 拐点提取：将路径节点序列转换为经纬度拐点坐标，便于前端绘制。

## 技术栈

| 技术 | 用途 |
|------|------|
| C++ | 核心业务逻辑 |
| Qt 6.x (Widgets + OpenGL) | GUI 框架、信号槽、事件处理 |
| OpenGL (固定管线) | 3D 场景渲染 |
| 最小二乘法 | 经纬度到地图像素坐标的仿射变换 |
| Dijkstra + 状态压缩 DP | 路径规划与多目标排序 |
| 耳切三角剖分 | 建筑轮廓网格生成 |
